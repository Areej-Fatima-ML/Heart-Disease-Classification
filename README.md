# Heart-Disease-Classification

Project Overview
Developed a Machine Learning model to classify whether a patient has heart disease based on 
medical attribtues.This project focuses on Binary Classification and model reliability.

Key Technical Steps

* EDA: Analyzed medical features like age, heart rate and correlation Heatmap for relationships between features.
* Preprocesssing: Handled missing data performed feature scaling.
* Modeling: Impelemented Logistic Regression
* Evaluation: Used Confusion Matrix ,Accuracy and Recall to ensure high prediction quality.

Tech Stack 
* Language : Python
* Libraries: Scikit-learn,Pandas,Seaborn,Matplotlib
